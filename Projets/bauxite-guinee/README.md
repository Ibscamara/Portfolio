# Analyse statistique de la production de bauxite en Guinée avec R

## Description
Ce projet porte sur l’étude de l’évolution de la production de bauxite en Guinée entre 2010 et 2023.

L’objectif est de montrer comment utiliser R pour :
- lire un fichier CSV,
- produire des statistiques descriptives,
- réaliser des graphiques,
- construire un modèle de régression linéaire,
- interpréter les résultats.

## Fichiers
- `bauxite_guinee.csv` : jeu de données
- `analyse_bauxite.R` : script principal
- `README.md` : présentation du projet

## Compétences mobilisées
- lecture et manipulation de données avec R
- statistique descriptive
- visualisation de données
- régression linéaire simple
- interprétation de résultats

## Idée générale
La Guinée est l’un des principaux producteurs mondiaux de bauxite. Ce projet permet d’étudier l’évolution de cette production sur plusieurs années à travers des outils simples de statistique et de modélisation.
